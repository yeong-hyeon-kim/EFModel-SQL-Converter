# EF Model SQL Converter

EF Model SQL Converter.

## Features

Convert EF Model to SQL Script Or SQL Script to EF Model.

## Extension Execution

1. Open The Ef Model Or SQL Script File.

2. Command pallet execution.
Ctrl + Shift + P

3. Use the Command "Create Script".
